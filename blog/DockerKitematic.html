How-to install Kitematic on Linux Ubuntu 16.04

Kitematic это инструмент с удобным графическим интерфейсом предназначенный для работы с контейнерами Docker. Официально, почему-то он для Linux не поставляется, хотя препятствий для работы на нашей ОС быть не может - ведь это веб-приложение. Давайте сейчас исправим эту несправедливость.


Вам нужно выполнить эти шаги:
  1. install docker
  2. sudo usermod -aG docker $USER
  3. reboot
  4. install NodeJS
  5. sudo apt-get install build-essential libcanberra-gtk-module
  6. git clone https://github.com/docker/kitematic.git
  7. cd kitematic
  8. make
  9. from kitematic folder, run 'npm start' (this will run the app)

P.S. Выполнив вход через приложение, вы столкнетесь с такой проблемой:
Невозможно скачать образ контейнера из Docker Hub.
Это легко решается - нужно выполнять вход используя не email, а ваш username.
